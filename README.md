# Cortex – AI Workplace Productivity Hub

## Project Overview

Cortex – AI Workplace Productivity Hub is a modern AI-powered SaaS web application designed to help professionals automate everyday workplace tasks. The platform combines multiple AI tools into one intuitive dashboard, allowing users to generate professional emails, summarize meeting notes, plan daily tasks, conduct research, and interact with an AI assistant.

The application is designed with a clean, responsive, and premium user interface inspired by leading SaaS platforms such as Linear, Notion AI, and Vercel. Cortex aims to improve workplace productivity by reducing repetitive work and helping users make better decisions faster.

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cortex-ai-workplace-productivity-hub.git
```

### 2. Navigate into the project

```bash
cd cortex-ai-workplace-productivity-hub
```

### 3. Install dependencies

```bash
npm install
```

### 4. Configure environment variables

Create a `.env` file in the project root and add your OpenAI API key:

```env
VITE_OPENAI_API_KEY=your_api_key_here
```

### 5. Start the development server

```bash
npm run dev
```

### 6. Open the application

Visit:

```
http://localhost:5173
```
